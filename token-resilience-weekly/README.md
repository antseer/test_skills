# token-resilience-weekly

代币韧性周报分析 -- 用相对强度筛选找出大盘下跌时最扛揍的币

## 这个工具做什么？

你手上盯着一堆代币，大盘跌了一周，你想知道：谁扛住了？谁被摁在地上摩擦？

这个工具帮你自动完成这个对比分析：

1. **拉基准线** -- 先算出 ETH 和 SOL 这一周每天平均跌了多少（比如 ETH 日均 -1.02%，SOL 日均 -1.08%）
2. **逐币对比** -- 把你的 watchlist 里每个代币的同期表现和基准比，算出谁跑赢、谁跑输
3. **韧性分级** -- 按表现分成四档：
   - **T1（正收益）**: 大盘跌它还涨，本周赢家
   - **T2（微跌但强于大盘）**: 跌了，但比 ETH/SOL 少跌很多
   - **弱势警示**: 跌幅是基准的 2-4 倍，资金在撤
   - **中性**: 表现和大盘差不多
4. **逐日标注** -- 不只看周均值，还会标出哪天暴涨、哪天扛跌，发现"跌时不跌、涨时跟上"的教科书级韧性模式

方法论来自 @Guolier8 的公开分析框架。

## 怎么用？

**基本用法：**

```
/token-resilience-weekly ["TAO","SPEC","KAITO","HYPE","IO","STORY","DRIFT"]
```

**参数说明：**

| 参数 | 必填 | 说明 | 默认值 |
|------|------|------|--------|
| watchlist | 是 | 要分析的代币列表 | -- |
| benchmarks | 否 | 基准代币 | ["ETH","SOL"] |
| time_range | 否 | 时间窗口 | "7d" |
| t1_threshold | 否 | T1 阈值（周均日收益 >= 此值为 T1） | 0.0 |
| t2_threshold_ratio | 否 | T2 阈值（跌幅 < 基准 x 此比例） | 0.5 |
| weak_threshold_ratio | 否 | 弱势阈值（跌幅 > 基准 x 此倍数） | 2.0 |

**自定义基准和阈值：**

```
/token-resilience-weekly ["TAO","KAITO","HYPE"] --benchmarks=["ETH","SOL","BTC"] --weak_threshold_ratio=3.0
```

## 什么时候用？什么时候不适合？

**适合的场景：**
- 每周日/周一做周报复盘，看看自己盯的币这周谁最强
- 大盘突然暴跌后，快速扫描哪些币扛住了
- 手上有一堆候选标的，想按相对强弱排个序
- 组合管理：识别持仓中的弱势标的，考虑是否换仓

**不适合的场景：**
- 想预测明天哪个币会涨 -- 韧性是回顾指标，不是水晶球
- 想分析为什么某个币抗跌（叙事归因需要人工判断）
- 想做短线交易信号 -- 这是周级别分析，不适合分钟/小时级操作
- watchlist 太小（2-3 个币）-- 分级意义不大，至少 5 个币以上效果更好
- 大盘横盘或上涨期 -- 韧性分析在大盘承压时最有价值

## 前置依赖

### 为什么需要装 MCP？

这个工具需要实时的市场价格数据才能工作。MCP（Model Context Protocol）服务器是数据来源 -- 把它理解成"给 Claude 接上数据管道"。不装的话，Claude 拿不到代币价格数据，分析就跑不起来。

### MCP 服务安装

#### Antseer Spot Market（现货市场数据）

**Claude Code (CLI)**
```bash
claude mcp add --transport http --scope user antseer-spot https://mcp.antseer.com/spot
```

**OpenClaw / Claw**

在设置页面添加 MCP 服务器，填写：
- 名称：`antseer-spot`
- URL：`https://mcp.antseer.com/spot`
- 传输类型：`http`

**OpenCode**

在配置文件 `opencode.json` 中添加：
```json
{
  "mcpServers": {
    "antseer-spot": {
      "type": "http",
      "url": "https://mcp.antseer.com/spot"
    }
  }
}
```

**通用 MCP 客户端**

任何支持 MCP 协议的客户端均可接入，核心参数：
- MCP 端点：`https://mcp.antseer.com/spot`
- 传输类型（Transport）：`http`
- 作用域（Scope）：`user`（推荐，跨项目共享）

安装完成后，**重启你的 Agent 客户端**以激活 MCP 服务。

# liquidity-pulse-cross-asset-predictor

流动性供应脉冲跨资产价格预测 — 用贵金属/大宗商品突破历史前高的脉冲参数，推算相关资产的峰值价格与达峰时间

---

## 这个工具做什么？

你在看市场时发现：白银刚刚突破了1980年的历史前高，正在快速拉升。与此同时，海力士（SK Hynix）也刚刚突破了1999年的历史前高。这两个资产是否处于同一"流动性脉冲"里？如果是，海力士可能涨到哪里？什么时候到顶？

这个工具帮你做三件事：

1. **量化参考资产的脉冲参数** — 白银从脉冲起点涨了多少倍？历时多少个交易日？
2. **验证目标资产的等价位置** — 海力士当前价格是否对应白银脉冲加速段的入口位置？
3. **推算目标资产的峰值和时间** — 按白银的脉冲幅度和时间节奏，海力士峰值在哪里？多少天后？

方法论来源：@xiaomustock 的"流动性供应脉冲跨资产映射"分析框架（2026年2月推文）。

---

## 怎么用？

直接告诉 Claude 你的参考资产和目标资产参数：

```
/liquidity-pulse-cross-asset-predictor
参考资产: 白银
ATH历史前高: $50 (1980年)
本轮新高: $120 (2026-01-29)
脉冲起点: $68.5 (2025-12-20)
目标资产: 海力士
目标ATH: 77万KRW (1999年)
目标当前价格: 107万KRW (2026-02-27)
```

或者用对话方式描述你的场景，Claude 会自动提取参数并运行推算。

### 参数说明

| 参数 | 是否必填 | 说明 |
|------|---------|------|
| 参考资产名称 | 必填 | 流动性领先的资产，如白银、黄金、BTC |
| 参考资产历史前高 | 必填 | 参考资产的长周期历史最高价（如1980年白银$50） |
| 参考资产本轮新高 | 必填 | 本轮刚突破的新高价格和日期 |
| 参考资产脉冲起点 | 必填 | 脉冲加速段的起始价格和日期（需技术分析判断） |
| 目标资产（可多个）| 必填 | 每个目标资产的名称、历史前高、当前价格和日期 |
| 交易日偏移量 | 可选，默认0 | 额外的时间窗口调整天数 |

## 什么时候用？什么时候不适合？

### 适合的场景

- 贵金属（白银、黄金）突破了几十年未见的历史前高，你想推算相关矿业股或半导体股的目标价
- 加密资产（BTC、ETH）突破历史ATH，你想推算处于相似位置的小市值代币或相关板块的脉冲幅度
- 参考资产和目标资产之间有明确的逻辑关联（同一流动性脉冲环境），需要量化时间窗口

### 不适合的场景

- 参考资产尚未突破历史前高（没有有效脉冲参数可供映射）
- 目标资产当前价格低于其历史前高（还在积累阶段，不在脉冲入口位置）
- 你想要止损价格或仓位建议（这个工具只做峰值预测，不做风险管理）
- 你需要实时韩国股票或中国A股ETF价格（MCP无法自动获取，需手动输入）

## 前置依赖

### 为什么需要装 MCP？

这个工具需要实时的链上/市场数据才能工作。MCP（Model Context Protocol）服务器是数据来源 — 把它理解成"给 Claude 接上数据管道"。不装的话，Claude 拿不到数据，分析就跑不起来。

### MCP 服务安装

#### ant-on-chain-mcp

**Claude Code (CLI)**
```bash
claude mcp add --transport http --scope user ant-on-chain-mcp https://ant-on-chain-mcp.antseer.ai/mcp
```

**OpenClaw / Claw**

在设置页面添加 MCP 服务器，填写：
- 名称：`ant-on-chain-mcp`
- URL：`https://ant-on-chain-mcp.antseer.ai/mcp`
- 传输类型：`http`

**OpenCode**

在配置文件 `opencode.json` 中添加：
```json
{
  "mcpServers": {
    "ant-on-chain-mcp": {
      "type": "http",
      "url": "https://ant-on-chain-mcp.antseer.ai/mcp"
    }
  }
}
```

**通用 MCP 客户端**

任何支持 MCP 协议的客户端均可接入，核心参数：
- MCP 端点：`https://ant-on-chain-mcp.antseer.ai/mcp`
- 传输类型（Transport）：`http`
- 作用域（Scope）：`user`（推荐，跨项目共享）

安装完成后，**重启你的 Agent 客户端**以激活 MCP 服务。

> 注意：此 Skill 的核心计算（脉冲参数推算）不依赖 MCP，可纯手动输入参数运行。MCP 主要用于辅助获取当前价格和技术指标作为交叉验证。

## Overview

以参考资产（如白银）突破历史前高后的流动性脉冲参数（倍幅、持续交易日）为模板，映射到处于相似流动性周期位置的目标资产（如半导体股票），推算目标资产峰值价格和达峰时间窗口。

## Action Specification

### Step 1: 计算参考资产脉冲参数

计算以下三个核心参数：

- **脉冲总倍幅** = `reference_new_ath_price` / `reference_ath_price`
  例：120 / 50 = 2.38x
- **脉冲加速段涨幅** = `reference_new_ath_price` / `reference_pulse_start_price`
  例：120 / 68.5 = 1.752x
- **脉冲加速段持续交易日** = `reference_new_ath_date` 与 `reference_pulse_start_date` 之间的交易日数
  例：2025-12-20 → 2026-01-29 = 26个交易日（排除周末）

### Step 2: 验证目标资产等价位置

对每个目标资产，验证其是否处于脉冲加速段入口：

- 目标资产相对历史前高涨幅 = (`target_current_price` - `target_ath_price`) / `target_ath_price`
- 等价映射到参考资产价格 = `reference_ath_price` × (1 + 目标涨幅)
- 验证：等价映射价格是否约等于 `reference_pulse_start_price`（偏差 < 15% 为有效）

如偏差 > 15%，降低置信度评级并在报告中注明。

### Step 3: 推算目标资产峰值价格

使用历史前高绑定法（精确公式，与原推文一致）：

- **目标资产峰值** = `target_ath_price` × (`reference_new_ath_price` / `reference_ath_price`)
  例：770000 × (120 / 50) = 1,848,000 KRW ≈ 183万

同时计算上涨空间：

- **上涨空间** = (峰值价格 - `target_current_price`) / `target_current_price` × 100%

### Step 4: 推算达峰时间

- 从 `target_current_date` 起，向后计算参考资产脉冲持续交易日数（排除周末和主要节假日）
- 达峰预计日期 = `target_current_date` + `pulse_duration_trading_days` 个交易日

### Step 5: 衍生资产联动推算（如有多个目标资产）

- 衍生资产峰值 = 衍生资产当前价格 × (主目标资产峰值 / 主目标资产当前价格)
- 保持相对强弱比率不变，分配相同时间窗口

### Step 6: 置信度评级

- **高**: 等价位置偏差 < 5%，且两资产处于相同宏观流动性环境（用户确认）
- **中**: 等价位置偏差 5%-15%，或相关性需进一步验证
- **低**: 等价位置偏差 > 15%，或资产间逻辑关联较弱

## Risk Parameters

- 此分析框架基于单一历史案例类比推演，样本量极小
- 韩国股票、中国A股ETF数据无法通过MCP自动获取，依赖用户手动输入
- 贵金属代币化价格（PAXG等）与实物白银期货现货价格可能存在溢价差异
- 流动性脉冲持续时间受宏观政策、黑天鹅事件等因素影响，时间窗口预测误差较大
- 此工具不提供止损建议、仓位管理建议，不构成投资建议

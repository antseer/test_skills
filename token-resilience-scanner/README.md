# token-resilience-scanner

代币韧性扫描器 — 用双向 beta 分析识别「大盘跌时最扛、涨时最猛」的山寨币

---

## 这个工具做什么？

你在大盘下行的一周里，想知道哪些代币「扛住了」——不只是跌得少，而是真正做到了大盘反弹时领涨、大盘下跌时抗跌的双向强势。

这个工具帮你：

1. **拉取数据**：通过 Antseer MCP 获取候选代币和基准代币（ETH/SOL）的每日收盘价序列
2. **计算双向超额**：区分大盘上涨日和下跌日，分别计算每个代币相对基准的超额收益（up_alpha + down_alpha）
3. **三档分层**：T1（周均正收益 + 综合韧性分为正）、T2（微跌但远优于基准）、弱势警示（跌幅超过基准 2× 以上）
4. **输出周报**：带异动日标注、叙事板块标签、下周重点观察建议的标准化 Markdown 表格

---

## 怎么用？

```
/token-resilience-scanner symbols=["TAO","KAITO","HYPE","IO","STORY","DRIFT"]
```

带完整参数：

```
/token-resilience-scanner symbols=["TAO","KAITO","HYPE","IO","STORY","DRIFT"] benchmark_symbols=["ethereum","solana"] time_range=7d end_date=2026-03-27 top_n=5 weak_n=3 min_daily_volume_usd=1000000
```

### 参数说明

| 参数 | 是否必填 | 默认值 | 说明 |
|------|---------|--------|------|
| `symbols` | 必填 | — | 待分析代币列表，支持 ticker（TAO）或 CoinGecko ID（bittensor） |
| `benchmark_symbols` | 可选 | ["ethereum","solana"] | 大盘基准代币，用于计算相对超额 |
| `time_range` | 可选 | "7d" | 分析时间窗口，目前支持 "7d"、"14d" |
| `end_date` | 可选 | 今日 | 分析截止日期，格式 YYYY-MM-DD |
| `top_n` | 可选 | 5 | T1 强势标的输出上限 |
| `weak_n` | 可选 | 3 | 弱势警示标的输出上限 |
| `min_daily_volume_usd` | 可选 | 1000000 | 流动性过滤阈值（美元），低于此值的代币跳过 |

## 什么时候用？什么时候不适合？

### 适合使用

- 每周收盘后（推荐周日或周一）定期生成一份市场相对强弱快照
- 大盘出现明显急跌后（BTC/ETH 单日 -5%+），快速定位「扛住了」的板块
- 构建下周观察列表时需要一个量化起点，而不是凭感觉选标的
- 你已经有一批固定关注的山寨币，想每周自动化排序

### 不适合使用

- 你想预测下周哪个币会涨——韧性是回顾性指标，T1 代币下周完全可能转弱
- 你的标的日均交易量 < 100 万美元——流动性不足会导致价格失真，信号不可靠
- 你需要自动构建候选池——watchlist 必须手动传入，本工具不负责选币
- 你需要仓位建议或具体止损位——本工具只做相对强弱排名，不给交易执行建议

## 前置依赖

### 为什么需要装 MCP？

这个工具需要实时的市场行情数据才能工作。MCP（Model Context Protocol，模型上下文协议）服务器是数据来源——把它理解成「给 Claude 接上行情数据管道」。不装的话，Claude 拿不到每日收盘价，韧性计算就跑不起来。

### MCP 服务安装

#### Antseer MCP（行情 + 情绪数据）

**Claude Code (CLI)**

```bash
claude mcp add --transport http --scope user antseer https://mcp.antseer.ai/mcp
```

**OpenClaw / Claw**

在设置页面添加 MCP 服务器，填写：
- 名称：`antseer`
- URL：`https://mcp.antseer.ai/mcp`
- 传输类型：`http`

**OpenCode**

在配置文件 `opencode.json` 中添加：

```json
{
  "mcpServers": {
    "antseer": {
      "type": "http",
      "url": "https://mcp.antseer.ai/mcp"
    }
  }
}
```

**通用 MCP 客户端**

任何支持 MCP 协议的客户端均可接入，核心参数：
- MCP 端点：`https://mcp.antseer.ai/mcp`
- 传输类型（Transport）：`http`
- 作用域（Scope）：`user`（推荐，跨项目共享）

安装完成后，**重启你的 Agent 客户端**以激活 MCP 服务。

### 本地脚本（可选）

如果想在本地测试计算逻辑（不依赖 MCP），可以直接运行：

```bash
python3 scripts/resilience_scanner.py
```

脚本内含模拟数据（2026.03.21–03.27 原始推文案例），会输出示例周报和 JSON 摘要。

## 关于数据精度

Antseer MCP 的 `sparkline` 接口返回的是近 7 天约 168 个小时级数据点，脚本取每 24 个点的最后一个作为「日收盘价」近似值。这与 CoinGecko 官方的每日 00:00 UTC 收盘价可能存在细微差异（通常 < 0.5%），不影响分层结论但会影响具体数值精度。报告中会标注「数据基于 sparkline 近似值」。

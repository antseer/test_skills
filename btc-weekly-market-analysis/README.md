# btc-weekly-market-analysis

BTC 周度综合市场分析 — 用宏观流动性 × ETF资金 × 衍生品结构 × 链上数据生成每周 BTC 市场研判报告

---

## 这个工具做什么？

你在每周开始时想知道："上周 BTC 发生了什么？现在该偏多还是偏空？"

这个工具帮你：

1. **采集七个维度的数据**：宏观利率/通胀、黄金联动、ETF资金流、衍生品结构、链上数据、价格技术位、恐慌贪婪指数
2. **逐层递进分析**：从外生宏观变量（Fed利率）到机构资金行为（ETF流向），再到市场结构（资金费率、OI），最后到链上数据（鲸鱼持仓、交易所存量）
3. **输出结构化周报**：包含信号汇总、支撑/压力位、短期方向判断、三情景路径（乐观/悲观/极端）和关键观察变量清单

分析方法论归属原作者 @Guolier8（推文：https://x.com/Guolier8/status/2039989943213404528）。

---

## 怎么用？

```
/btc-weekly-market-analysis BTC
/btc-weekly-market-analysis BTC --start_date=2026-03-22 --end_date=2026-03-28
/btc-weekly-market-analysis BTC --start_date=2026-03-22 --end_date=2026-03-28 --include_eth_etf=true --whale_threshold=1000 --fear_greed_threshold=20
```

**参数说明：**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| symbol | string | 是 | BTC | 分析标的，当前主要支持 BTC |
| start_date | string | 否 | 上周一 | 分析开始日期，格式 YYYY-MM-DD |
| end_date | string | 否 | 上周日 | 分析结束日期，格式 YYYY-MM-DD |
| include_eth_etf | boolean | 否 | true | 是否同时分析 ETH ETF 资金流（作为市场情绪参考） |
| whale_threshold | integer | 否 | 1000 | 鲸鱼地址定义阈值（持有 BTC 数量） |
| whale_lookback_days | integer | 否 | 14 | 鲸鱼净增减持统计时间窗口（天） |
| fear_greed_threshold | integer | 否 | 20 | 极端恐慌阈值，低于此值时附加历史统计概率 |

不填 start_date / end_date 时，自动分析上一周（周一至周日，UTC时区）。

## 什么时候用？什么时候不适合？

**适合：**
- 每周开始时快速了解上周 BTC 市场全貌
- 宏观事件（Fed 决议、CPI 公布）后更新市场判断
- 需要多维度信号汇总而不想手动逐一查看各数据平台
- 投研报告底稿生成（再由分析师修订）

**不适合：**
- 日内交易决策（这是周度框架，不适合小时级别操作）
- 山寨币分析（当前仅针对 BTC 调优，其他资产逻辑不同）
- 期权策略制定（期权 put/call 偏斜数据无 MCP 覆盖）
- 精确价格预测（框架给出方向和区间，不给具体入场价格）
- 无法预测黑天鹅事件（监管突发、交易所暴雷等）

## 前置依赖

### 为什么需要装 MCP？

这个工具需要实时的宏观数据、ETF 资金流、链上数据等多源数据才能工作。MCP（Model Context Protocol）服务器是数据来源 — 把它理解成"给 Claude 接上多条数据管道"。不装的话，Claude 拿不到数据，分析就跑不起来。

### MCP 服务安装

#### Antseer MCP 服务

本工具调用以下 Antseer MCP 工具（需要一个 Antseer MCP 服务端点）：

- `ant_macro_economics` — Fed 利率、CPI 数据
- `ant_precious_metal_tokens` — 黄金 token 价格（PAXG）
- `ant_etf_fund_flow` — BTC/ETH ETF 资金流
- `ant_futures_market_structure` — 资金费率、OI、多空比
- `ant_futures_liquidation` — 爆仓数据
- `ant_fund_flow` — 交易所存量/净流量
- `ant_stablecoin` — 稳定币市值
- `ant_token_analytics` — 鲸鱼持仓、MVRV
- `ant_spot_market_structure` — BTC 价格/K线
- `ant_market_sentiment` — Fear & Greed 指数

**Claude Code (CLI)**
```bash
claude mcp add --transport http --scope user antseer <antseer-mcp-server-url>
```

**OpenClaw / Claw**

在设置页面添加 MCP 服务器，填写：
- 名称：`antseer`
- URL：`<antseer-mcp-server-url>`
- 传输类型：`http`

**OpenCode**

在配置文件 `opencode.json` 中添加：
```json
{
  "mcpServers": {
    "antseer": {
      "type": "http",
      "url": "<antseer-mcp-server-url>"
    }
  }
}
```

**通用 MCP 客户端**

任何支持 MCP 协议的客户端均可接入，核心参数：
- MCP 端点：`<antseer-mcp-server-url>`
- 传输类型（Transport）：`http`
- 作用域（Scope）：`user`（推荐，跨项目共享）

安装完成后，**重启你的 Agent 客户端**以激活 MCP 服务。

### 无 MCP 降级模式

若无法安装 MCP，工具仍可运行，但以下数据将缺失（报告中标注"数据暂不可用"）：
- 实时 ETF 资金流数据（建议手动查阅 farside.co.uk）
- 链上鲸鱼数据（建议查阅 CryptoQuant）
- 实时 Fear & Greed 指数（建议查阅 alternative.me）

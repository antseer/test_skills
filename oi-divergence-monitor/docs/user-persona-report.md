## Skill 功能摘要

| 维度 | 内容 |
|------|------|
| 核心功能 | 检测加密货币现货价格与永续合约 OI 之间的背离关系，通过资金费率、爆仓数据、多空比三维度交叉验证，输出量化背离评分和结构化报告 |
| 输入/输出 | 输入：代币符号（必填）+ 可选阈值参数；输出：信号等级（strong/moderate/weak/none）、背离评分（0-100）、结构化分析报告 |
| 依赖 MCP 工具 | ant_spot_market_structure, ant_futures_market_structure, ant_futures_liquidation |
| 知识门槛 | **中高** -- 需理解 OI（未平仓量）、资金费率、爆仓机制、多空比等永续合约核心概念 |
| 操作门槛 | **低** -- 最简场景仅需输入代币符号即可运行，默认参数适用于主流币种 |
| 使用频率 | **中高频** -- 定期巡检（每 4-8 小时）+ 价格异动事件驱动 |
| 决策类型 | 交易风控决策 -- 判断价格上涨是否可持续，辅助减仓/对冲时机选择 |

## 优化建议

1. **降低解读门槛** -- 当前输出需要用户理解 OI、资金费率等概念 --> 增加 summary 模式，用一句通俗语言概括结论（如"当前 BTC 上涨缺乏杠杆支撑，回调风险较高"） --> 覆盖更多中级交易者，预计目标用户群扩大 50-80%

2. **主动监控模式** -- 当前定期巡检依赖用户手动触发 --> 支持定时自动运行，strong 信号时主动推送告警 --> 提升用户粘性，信号捕获率从被动 ~60% 提升至 ~95%

3. **历史信号回溯** -- 用户无法评估信号历史命中率 --> 增加 history 子命令，维护本地信号记录并回顾后续价格表现 --> 帮助用户建立信号有效性的直觉，增强信任度和续用率

4. **批量扫描模式** -- 多币种对比需逐个运行 --> 支持 /oi-divergence-monitor BTC,ETH,SOL 批量输出对比表 --> 板块轮动分析效率提升 N 倍

5. **新手引导** -- OI 背离是专业话题，大众认知度低 --> 首次使用时提供 30 秒教学引导，输出中嵌入关键指标简要解释 --> 降低新用户学习曲线，减少流失率

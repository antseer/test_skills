## Skill 功能摘要

| 维度 | 内容 |
|------|------|
| 核心功能 | 根据热点叙事关键词扫描链上匹配代币，通过社交热度、持仓分布、Smart Money、交易活跃度四维评估，判断叙事生命周期阶段，输出综合信号评分和操作建议 |

| 使用频率 | 事件驱动型，高频用户每周 5-10 次，低频用户每周 1-2 次 |
| 决策类型 | 交易决策（进场时机判断）+ 研究分析（叙事生命周期评估） |

## 优化建议

1. **热门叙事推荐功能** -- 当前叙事关键词需用户自行输入，是全流程最大瓶颈 -> 增加 topics_list 热门话题展示，将开放式输入降级为选择题 -> 预计覆盖不知该搜什么的中低经验用户，扩大 TAM 30-50%

2. **快速模式（速度优先）** -- 6 步完整分析耗时 30-60 秒，与 MEME 交易秒级时效矛盾 -> 增加仅执行 Step 1-2 的快速模式，5-10 秒返回初筛结果，可对单个代币触发深度分析 -> 响应时间缩短 70-80%，贴近原推文速度优先方法论

3. **一句话结论先行** -- 6 维度评分对新手信息量过大 -> 表格前增加一行结论（如"ALIENS 是当前最优候选"），weak 信号默认折叠 -> 降低输出解读门槛，新手无需理解各维度即可获得可操作结论

4. **语义扩展搜索** -- 文本匹配可能遗漏名称不含关键词的叙事代币 -> 对输入关键词自动生成 2-3 个同义/关联词并行搜索 -> 提升代币匹配召回率 20-30%

5. **叙事监控模式** -- 当前为被动查询，缺乏主动触发场景 -> 用户预设叙事主题列表，定时检查热度变化并推送告警 -> 将使用模式从被动查询升级为主动监控，增加用户粘性

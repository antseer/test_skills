Core Metrics
指标解释
YES Price — Polymarket 上"是"的价格（0-1），代表市场隐含概率。3.9¢ = 市场认为 3.9% 概率发生。
True Prob — 你自己估算的真实概率。与市场价差越大 = Edge 越大。
EV (期望值) — 每 $1 投入的预期收益。公式：P×(1/价格-1)-(1-P)。正值 = 值得下注。
Edge — 你的真实概率 vs 市场隐含概率的差值。正 Edge = 你比市场更准。

Four-Condition Survival Check
四条件验证
正期望值 — EV > 0 才值得下注。EV ≤ 0 = 长期必亏。
Kelly 仓位 — 用 Kelly 公式控制仓位大小，防止单笔亏太多。1/4 Kelly 更保守。
模型驱动 — 需要有量化模型支撑，不能靠"我觉得"。
结构性优势 — 你有别人没有的信息/能力（专有数据、自动化系统等）。

Kelly Position Sizing
Kelly 公式
Full Kelly = (p×b − (1−p)) / b，其中 p = 真实概率，b = 赔率。数学最优但波动大。
1/4 Kelly — 只下 Full Kelly 的 25%，大幅降低破产风险。保险模型推荐 1/4 Kelly。
上限 5% — 单笔不超过总资金 5%。即使 Kelly 算出更多，也卡在 5% 防止爆仓。

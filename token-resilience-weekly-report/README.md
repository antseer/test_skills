# token-resilience-weekly-report

代币韧性周报 — 用双向贝塔不对称性识别大盘承压周内「涨时超涨、跌时超抗跌」的代币，按 T1/T2/弱势/中性分级并生成结构化周报。

---

## 这个工具做什么？

你在每周大盘下跌后想知道：**这一周哪些代币真正扛住了？哪些代币是"真韧性"还是假抗跌？**

这个工具帮你做三件事：

1. **量化"韧性"**：不只看谁的周均收益高，而是分别计算「大盘上涨日该代币超涨多少」（upside_alpha）和「大盘下跌日该代币少跌多少」（downside_alpha），两者之和就是韧性得分。这才是真正的双向超额表现。

2. **自动分级**：T1（涨跌双向均跑赢 + 周均正收益）→ T2（至少一侧跑赢）→ 弱势警示（跌幅超基准若干倍）→ 中性。排名规则透明，参数可调。

3. **生成周报**：输出含逐日涨跌幅矩阵、关键日期标注、一句话结论的完整 Markdown 周报，可直接复制发送。

---

## 怎么用？

### 基本调用

```
/token-resilience-weekly-report TAO SPEC KAITO HYPE IO STORY DRIFT
```

### 参数说明

| 参数 | 是否必填 | 默认值 | 说明 |
|------|----------|--------|------|
| symbols（位置参数） | 是 | — | 待分析的代币 ticker，空格分隔，如 `TAO KAITO HYPE` |
| benchmarks | 否 | BTC, ETH, SOL | 市场基准代币，用于计算上涨日/下跌日 |
| week_start | 否 | 上周一 | 分析周开始日期，格式 `2026-03-21` |
| week_end | 否 | 上周日 | 分析周结束日期，格式 `2026-03-27` |
| t1_min_return | 否 | 0.0 | T1 门槛：周均日收益最低值（%），调高可收紧 T1 标准 |
| t2_min_return | 否 | -0.5 | T2 门槛：允许的最小周均日收益（%） |
| weak_multiplier | 否 | 2.0 | 弱势警示门槛：跌幅超过基准多少倍触发警示 |
| include_narrative | 否 | false | 是否为 T1 代币生成叙事催化剂注释（需额外调用 MCP） |

### 指定时间范围示例

```
/token-resilience-weekly-report TAO KAITO HYPE IO --week_start=2026-03-21 --week_end=2026-03-27
```

## 什么时候用？什么时候不适合？

### 适合

- 每周一复盘上周，快速识别本轮下跌中谁在扛
- 大盘单周跌幅超过 5% 后，事件驱动评估组合中各标的韧性
- 对比多个板块代币（如 AI、DePIN、DeFi），识别板块内相对强弱
- 更新自己的 watchlist，淘汰持续弱势代币，关注韧性信号

### 不适合

- 预测某代币下周涨跌（韧性是描述性历史指标，不是预测指标）
- 分析流动性极低的长尾代币（价格数据可能因单笔交易失真）
- 需要精确到分钟级的高频交易决策（sparkline 日收盘价为近似值）
- 替代深度基本面分析（韧性只反映价格行为，不解释原因）
- 在 BTC 单周剧烈波动（>15%）的极端行情中（sparkline 误差可能扩大）

## 前置依赖

### 为什么需要装 MCP？

这个工具需要实时的加密货币市场数据才能工作。MCP（Model Context Protocol）服务器是数据来源 — 把它理解成"给 Claude 接上市场数据管道"。不装的话，Claude 拿不到价格数据，韧性计算就跑不起来。

### MCP 服务安装

#### Antseer MCP 服务

**Claude Code (CLI)**
```bash
claude mcp add --transport http --scope user antseer https://mcp.antseer.io/mcp
```

**OpenClaw / Claw**

在设置页面添加 MCP 服务器，填写：
- 名称：`antseer`
- URL：`https://mcp.antseer.io/mcp`
- 传输类型：`http`

**OpenCode**

在配置文件 `opencode.json` 中添加：
```json
{
  "mcpServers": {
    "antseer": {
      "type": "http",
      "url": "https://mcp.antseer.io/mcp"
    }
  }
}
```

**通用 MCP 客户端**

任何支持 MCP 协议的客户端均可接入，核心参数：
- MCP 端点：`https://mcp.antseer.io/mcp`
- 传输类型（Transport）：`http`
- 作用域（Scope）：`user`（推荐，跨项目共享）

安装完成后，**重启你的 Agent 客户端**以激活 MCP 服务。

### 本工具使用的 MCP 工具

| 工具名 | 用途 | 覆盖度 |
|--------|------|--------|
| ant_spot_market_structure (coins_markets + sparkline_7d) | 获取基准和目标代币 7 日价格数据 | 覆盖（近似日收盘价） |
| ant_meme (search_pairs) | 未知 ticker 查询 CoinGecko ID | 完整覆盖 |
| ant_market_sentiment (coin_detail) | 可选：T1 代币叙事催化剂识别 | 完整覆盖 |
